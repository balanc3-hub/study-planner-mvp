# Study Planner MVP

## Пітч
Study Planner — це простий сервіс для студентів, який допомагає швидко додавати навчальні завдання та відмічати їх виконання. Продукт зменшує плутанину в навчанні та допомагає тримати фокус на актуальних задачах.

## Проблема
Студенти часто зберігають навчальні завдання в різних місцях, через що забувають дедлайни та губляться у пріоритетах.

## Ціль першої ітерації
Перевірити, чи достатньо простого списку завдань для зручного контролю навчального процесу.

## MVP
Мінімальна версія продукту включає:
- додавання навчального завдання;
- відображення списку завдань;
- можливість позначити завдання як виконане.

## Стейкхолдери
Основний стейкхолдер — студент. Другорядні — викладач та розробник навчального проєкту.

## Критерії успіху
- користувач може додати та виконати завдання менш ніж за 1 хвилину;
- головний сценарій проходиться без помилок;
- новий користувач розуміє інтерфейс без пояснень.
